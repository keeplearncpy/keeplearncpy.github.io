# 一、类型别名与自定义类型

# 1 类型别名

类型别名是 Go 1.9 版本添加的新功能，主要用于代码升级和迁移。类型别名规定：TypeAlias 只是 Type 的别名，本质上 TypeAlias 与 Type 是同一个类型，就像一个孩子小时候有小名、乳名，上学后用学名，英语老师又会给他起英文名，但这些名字都指的是他本人。

举例：

    type TypeAlias = Type // 类型别名
1

# 2 自定义类型

自定义类型是定义了一个全新的类型。我们可以基于内置的基本类型定义，也可以通过 struct 定义。

举例：

    type NewInt int
    // type IntAlias = int // 类型别名
1
2

两者区别：

  1. NewInt 类型和 int 类型虽然都是 int 类型，但是 Go 语言认为 NewInt 类型和 int 类型是两种不同的类型，编译器也不会把 NewInt 类型的变量当做 int 类型的变量来处理。
  2. IntAlias 类型只会在代码中存在，编译完成时，不会有 IntAlias 类型。内置的 byte、rune 等都是类型别名。

# 二、结构体

# 1. 结构体的定义

Go 语言中，结构体（struct）是一种自定义的数据类型，可以封装多个基本类型的变量。结构体是由一系列具有相同类型或不同类型的数据构成的数据集合。

代码格式：

    type 类型名 struct {
        字段名 字段类型
        字段名 字段类型
        …
    }
1
2
3
4
5

类型名：标识自定义结构体的名称，在同一个包内不能重复。 字段名：表示结构体字段名。结构体中的字段名必须唯一。 字段类型：表示结构体字段的具体类型。

举例：

    type person struct {
        name string
        age int
    }

# 3. 结构体实例化

Go 语言中，结构体本身是一种类型，可以像整型、字符串等类型一样，以 var 的方式声明结构体即可完成实例化。

举例：

    // 1. 基本实例化
    var p1 person
    p1.name = "小王子"
    p1.age = 18
    fmt.Printf("p1=%v\n", p1) // p1={小王子 18}
    fmt.Printf("p1=%#v\n", p1) // p1=main.person{name:"小王子", age:18}

    // 2. 使用键值对初始化
    p2 := person{
        name: "小王子",
        age:  28,
    }

    // 3. 使用new关键字对结构体进行实例化，得到的是结构体的地址
    p3 := new(person)
    // 因为p3是指针，因此标准的给字段赋值方式
    // (*p3).name = "小王子" 也可以这样写 p3.name = "小王子"
    p3.name = "小王子"
    p3.age = 28
    fmt.Printf("p3 type is %T\n", p3) // p3 type is *main.person

    // 4. 取结构体的地址实例化
    p4 := &person{}
    fmt.Printf("p4 type is %T\n", p4) // p4 type is *main.person

# 5. 结构体内存布局

结构体占用一块连续的内存。

举例：

    type test struct {
        a int8
        b int8
        c int8
        d int8
    }

    n := test{
        a: 1,
        b: 2,
        c: 3,
        d: 4,
    }
    fmt.Printf("n.a %p\n", &n.a) // n.a 0xc00000a0b0
    fmt.Printf("n.b %p\n", &n.b) // n.b 0xc00000a0b1
    fmt.Printf("n.c %p\n", &n.c) // n.c 0xc00000a0b2
    fmt.Printf("n.d %p\n", &n.d) // n.d 0xc00000a0b3

# 6. 使用构造函数创建结构体实例

Go 语言的结构体没有构造函数，我们可以自己实现。

package main

import "fmt"

//  构造函数：约定成俗用new开头
//  构造函数 创建结构体变量的函数

type person struct {
	name string
	age  int
}

type person1 struct {
	name    string
	age     int
	addr    string
	phone   string
	habbies []string
	//.....

}

// 返回的是结构体还是结构体指针
// 当结构体比较大的时候尽量使用结构体指针，减少程序的内存开销
// 当结构体中有指针的时候，不能使用结构体指针，因为指针的指针会导致内存泄漏
func newPerson(name string, age int) person {
	return person{
		name: name,
		age:  age,
	}
}

// 结构体比较大时，使用结构体指针
func newPerson1(name string, age int, addr string, phone string) *person1 {
	return &person1{
		name:  name,
		age:   age,
		addr:  addr,
		phone: phone,
	}
}

func main() {
	p1 := newPerson("张三", 18)
	p2 := newPerson1("李四", 20, "北京市", "123456789")
	fmt.Println(p1)
	fmt.Println(p2)

}

# 7. 方法和接受者

# 1. 什么是方法，方法的定义

Go 语言中的方法（Method）是一种作用于特定类型变量的函数。这种特定类型变量叫做接收者（Receiver）。接收者的概念就类似于其他语言中的 this 或者 self。

方法的定义格式如下：

    func (接收者变量 接收者类型) 方法名(参数列表) (返回参数) {
        函数体
    }
    fun (p person) dream(str string) {
        fmt.Printf("%s的梦想是学好Go语言！\n", p.name)
    }
1
2
3
4
5
6

其中，

  1. 接收者变量：接收者中的参数变量名在命名时，官方建议使用接收者类型名称首字母的小写，而不是 self、this 之类的命名。例如，Person 类型的接收者变量应该命名为 p，Connector 类型的接收者变量应该命名为 c 等。
  2. 接收者类型：接收者类型和参数类似，可以是指针类型和非指针类型。
  3. 方法名、参数列表、返回参数：具体格式与函数定义相同。

# 2. 方法与函数的区别

  1. 调用方式不同

    函数调用方式：函数名(参数列表)

    方法调用方式：变量.方法名(参数列表)

  2. 对于普通函数，接收者为值类型时，不能将指针类型的数据直接传递，反之亦然。

    对于方法（如 struct 的方法），接收者为值类型时，可以直接用指针类型的变量调用方法，反过来同样也可以。

# 3. 方法的调用

    package main

    import "fmt"

    type person struct {
        name string
        age  int
    }

    // 构造函数
    func newPerson(name string, age int) *person {
        return &person{
            name: name,
            age:  age,
        }
    }

    // 方法
    // 接收者使用值类型：传拷贝进去
    func (p person) dream(str string) {
        fmt.Printf("%s的梦想是学好Go语言！\n", p.name)
    }

    // 接收者使用指针类型：传地址进去
    func (p *person) setAge(newAge int) {
        p.age = newAge
    }

    func main() {
        p1 := newPerson("张三", 18)
        p1.dream("做个咸鱼")
        fmt.Println(p1.age) // 18
        p1.setAge(28)
        fmt.Println(p1.age) // 28
    }

# 8. 结构体的匿名字段

# 1. 什么是匿名字段

结构体允许其成员字段在声明时没有字段名而只有类型，这种没有名字的字段就称为匿名字段。

使用场景：当我们需要定义一个结构体，它只包含一个字段时，可以考虑使用匿名字段。

字段比较少也比较简单的时候可以使用。不常用！！！

# 2. 匿名字段的使用

    package main

    import "fmt"

    type person struct {
        string
        int
    }

    func main() {
        p1 := person{
            "小王子",
            18,
        }
        fmt.Println(p1) // {小王子 18}
        fmt.Println(p1.string) // 小王子
        fmt.Println(p1.int) // 18
    }

# 9. 自定义类型

# 1. 什么是自定义类型

Go 语言中可以使用 type 关键字来定义自定义类型。例如：

    type myInt int // 自定义类型
1

# 2. 自定义类型的使用

    package main

    import "fmt"

    type myInt int // 自定义类型
    // 给自定义类型添加方法
    // 不能给别的包里面的类型添加方法，只能给自己包里面的类型添加方法
    // 可以自己定义类型，然后给自己定义的类型添加方法
    func (m myInt) hello() {
	    fmt.Println("自定义类型添加的方法")
    }

    func main() {
	// 声明一个myInt类型的变量m,值为100
	    m := myInt(100)
	    m.hello()        //    自定义类型添加的方法
	    fmt.Println(m)  // 100
    }

# 10. 结构体的继承

Go 语言中使用结构体也可以实现其他编程语言中面向对象的继承。

Go 语言中的继承是通过匿名字段来实现的。

# 1. 什么是结构体的继承

结构体可以使用匿名字段，也可以使用非匿名字段。匿名字段就是没有字段名，只有字段类型。

# 2. 如何实现继承

例如：

    package main

    import "fmt"

    /**
        结构体实现继承
    */

    type animal struct {
	    name string
    }

    func (a animal) run() {
	    fmt.Println("动物会跑")
    }

    type dog struct {
	    age    int
	    animal // 通过匿名结构体实现继承
    }

    func (d dog) jeep() {
	    fmt.Println("狗会跳", d.age)
    }

    func main() {
	    d1 := dog{
		    age:    10,
		    animal: animal{name: "旺财"},
	    }
	    fmt.Println(d1)
	    d1.run()
	    d1.jeep()
    }

# 11. 结构体与 json 序列化

# 1. 什么是 json 序列化

JSON(JavaScript Object Notation) 是一种轻量级的数据交换格式。易于人阅读和编写。同时也易于机器解析和生成。它基于 JavaScript Programming Language, Standard ECMA-262 3rd Edition - December 1999 的一个子集。JSON 采用完全独立于语言的文本格式，但是也使用了类似于 C 语言家族的习惯（包括 C、C++、C#、Java、JavaScript、Perl、Python 等）。这些特性使 JSON 成为理想的数据交换语言。

# 2. 结构体与 json 序列化

Go 语言中内置的 encoding/json 包可以将 Go 语言中的结构体变量 --> json 格式的字符串。

# 3. 结构体与 json 序列化的使用

    package main

    import (
        "encoding/json"
        "fmt"
    )

    type person struct {
        // 字段名首字母必须大写 否则无法访问 如果是小写的话，json包访问不到
        Name string `json:"name" db:"name" ini:"name"` // 通过反射机制获取tag标签的值
        Age  int    `json:"age"`
    }

    func main() {
        p1 := person{
            Name: "小王子",
            Age:  18,
        }
        // 序列化
        b, err := json.Marshal(p1)
        if err != nil {
            fmt.Printf("marshal failed, err:%v\n", err)
            return
        }
        fmt.Printf("%#v\n", string(b)) // "{\"name\":\"小王子\",\"age\":18}"
        //  去掉转义字符
        fmt.Printf("%s\n", b) // {"name":"小王子","age":18}

        // 反序列化
        str := `{"name":"小王子","age":18}`
        var p2 person
        json.Unmarshal([]byte(str), &p2)
        fmt.Printf("%#v\n", p2) // main.person{Name:"小王子", Age:18}
    }

# 12. 结构体嵌套

# 1. 什么是结构体嵌套

Go 语言中的结构体中可以包含一个或多个匿名或非匿名字段，即结构体嵌套。

# 2. 结构体嵌套的使用

    package main

    import "fmt"

    type address struct {
        province string
        city     string
    }

    type workPlace struct {
        province string
        city     string
    }

    type person struct {
        name    string
        age     int
        address // 匿名嵌套结构体
        workPlace
    }

    func main() {
        p1 := person{
            name: "小王子",
            age:  18,
            address: address{
                province: "山东",
                city:     "威海",
            },
            workPlace: workPlace{
                province: "上海",
                city:     "浦东",
            },
        }
        fmt.Println(p1) // {{小王子 18} {上海 浦东}}
        fmt.Println(p1.address.city) // 威海
        // 如何直接拿到p1.address
	    // 可以通过匿名结构体来实现  workPlace: workPlace
        fmt.Println(p1.city) // 威海
    }

# 13. 结构体练习

实现一个学生管理系统，包含增删改查

package main

import (
	"fmt"
	"os"
)

/*
*
结构体练习
---学生管理系统
*/

type student struct {
	id   int64
	name string
}

// 造一个学生的管理者
type studentMgr struct {
	allStudent map[int64]student
}

// 查看所有学生
func (s studentMgr) showStudent() {
	// 从s.allStudent这个map中把所有的学生打印出来
	for _, v := range s.allStudent {
		fmt.Printf("学号：%d 姓名：%s\n", v.id, v.name)
	}
}

// 增加学生
func (s studentMgr) addStudent() {
	// 1. 根据用户输入的内容创建一个新的学生
	var (
		stuID   int64
		stuName string
	)
	// 获取用户输入
	fmt.Print("请输入学号：")
	fmt.Scanln(&stuID)
	fmt.Print("请输入姓名：")
	fmt.Scanln(&stuName)
	newStudent := student{
		id:   stuID,
		name: stuName,
	}
	// 2. 把新的学生放到s.allStudent这个map中
	s.allStudent[newStudent.id] = newStudent
	fmt.Println("添加成功")

}

// 修改学生
func (s studentMgr) editStudent() {
	// 1. 获取用户输入的学号
	fmt.Print("请输入学号：")
	var stuID int64
	fmt.Scanln(&stuID)
	// 2. 展示该学号对应的学生信息，如果没有提示查无此人
	stuObj, ok := s.allStudent[stuID]
	if !ok {
		fmt.Print("查无此人")
		return
	}
	fmt.Printf("你要修改的学生信息如下：学号：%d 姓名：%s\n", stuObj.id, stuObj.name)
	// 3. 请输入修改后的学生名
	fmt.Print("请输入学生的新名字：")
	var newName string
	fmt.Scanln(&newName)
	// 4. 更新学生的姓名
	stuObj.name = newName
	s.allStudent[stuID] = stuObj
	fmt.Println("修改成功")

}

// 删除学生
func (s studentMgr) deleteStudent() {
	// 1. 请用户输入要删除的学生的学号
	fmt.Print("请输入要删除的学生的学号：")
	var stuID int64
	fmt.Scanln(&stuID)
	// 2. 去map找有没有这个学生，如果没有打印查无此人的提示
	_, ok := s.allStudent[stuID]
	if !ok {
		fmt.Println("查无此人")
		return
	}
	// 3. 有的话就删除
	delete(s.allStudent, stuID)
	fmt.Println("删除成功")

}

// 声明全局变量-----所有学生对象
var smr studentMgr

// 菜单函数
func showMenu() {
	fmt.Println("欢迎来到学生管理系统")
	fmt.Println(`
	1. 查看所有学生
	2. 新增学生
	3. 修改学生
	4. 删除学生
	5. 退出系统
	`)
}

func main() {
	// 初始化学生管理对象
	smr = studentMgr{
		allStudent: make(map[int64]student, 100),
	}
	for {
		showMenu()
		// 等待用户输入选项
		fmt.Print("请输入选项：")
		var choice int
		fmt.Scanln(&choice)
		fmt.Println("你输入的是：", choice)

		switch choice {
		case 1:
			smr.showStudent()
		case 2:
			smr.addStudent()
		case 3:
			smr.editStudent()
		case 4:
			smr.deleteStudent()
		case 5:
			os.Exit(1)
		default:
			fmt.Println("请输入正确的选项")
		}
	}
}
