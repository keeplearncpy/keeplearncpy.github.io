# Go 语言笔记

# 一、简介与基本语法

# 1. 简介

Go 语言是一种开源的静态类型编程语言，它具有简洁的语法和良好的并发支持。

# 1. 基本结构

  1. Go 语言的基本结构

# 3. 基本数据类型

  1. 整型：

int：有符号整数类型，大小和平台相关（32 位或 64 位）。 int8、int16、int32、int64：有符号整数类型，分别占用 8、16、32、64 位。 uint：无符号整数类型，大小和平台相关。 uint8、uint16、uint32、uint64：无符号整数类型，分别占用 8、16、32、64 位。 uintptr：用于存放指针的整数类型。

  1. 浮点型：

float32：单精度浮点数类型，占用 32 位。 float64：双精度浮点数类型，占用 64 位。 complex64：复数类型，由两个 float32 构成。 complex128：复数类型，由两个 float64 构成。

package main

import (
	"fmt"
	"math"
	"unsafe"
)

func main() {
	var (
		i8   int8
		i16  int16
		i32  int32
		i64  int64
		u8   uint8
		u16  uint16
		u32  uint32
		u64  uint64
		f32  float32
		f64  float64
		c64  complex64
		c128 complex128
	)
	fmt.Printf("%T %dB %v~%v\n", i8, unsafe.Sizeof(i8), math.MinInt8, math.MaxInt8)
	fmt.Printf("%T %dB %v~%v\n", i16, unsafe.Sizeof(i16), math.MinInt16, math.MaxInt16)
	fmt.Printf("%T %dB %v~%v\n", i32, unsafe.Sizeof(i32), math.MinInt32, math.MaxInt32)
	fmt.Printf("%T %dB %v~%v\n", i64, unsafe.Sizeof(i64), math.MinInt64, math.MaxInt64)
	fmt.Printf("%T %dB %v~%v\n", u8, unsafe.Sizeof(u8), 0, math.MaxUint8)
	fmt.Printf("%T %dB %v~%v\n", u16, unsafe.Sizeof(u16), 0, math.MaxUint16)
	fmt.Printf("%T %dB %v~%v\n", u32, unsafe.Sizeof(u32), 0, math.MaxUint32)
	fmt.Printf("%T %dB %v~%v\n", u64, unsafe.Sizeof(u64), 0, uint64(math.MaxUint64))
	fmt.Printf("%T %dB %v~%v\n", f32, unsafe.Sizeof(f32), math.SmallestNonzeroFloat32, math.MaxFloat32)
	fmt.Printf("%T %dB %v~%v\n", f64, unsafe.Sizeof(f64), math.SmallestNonzeroFloat64, math.MaxFloat64)
	fmt.Printf("%T %dB %v~%v\n", c64, unsafe.Sizeof(c64), complex(math.SmallestNonzeroFloat32, math.SmallestNonzeroFloat32), complex(math.MaxFloat32, math.MaxFloat32))
	fmt.Printf("%T %dB %v~%v\n", c128, unsafe.Sizeof(c128), complex(math.SmallestNonzeroFloat64, math.SmallestNonzeroFloat64), complex(math.MaxFloat64, math.MaxFloat64))
}

  1. 布尔型：

bool：布尔类型，取值为 true 或 false。

  1. 字符串类型：

string：字符串类型，由一串字节组成。

  1. 字符类型：

  1. 数组类型：

array：固定长度的数组，元素类型和长度在声明时指定。 切片类型：

pointer：指向变量内存地址的数据类型。

  1. 十进制、八进制、十六进制：
package main

	"fmt"

func main() {
    var a int = 10
    var b int = 010 //八进制，以0开头

}

# 5. 基本语法---包的声明

在 Go 语言中，所有的包名都应该使用小写字母。

在 Go 语言中，你必须在源文件中非注释的第一行指明这个文件属于哪个包，如：package main。package main 表示一个可独立执行的程序，每个 Go 应用程序都包含一个名为 main 的包。

一个应用程序可以包含不同的包，而且即使你只使用 main 包也不必把所有的代码都写在一个巨大的文件里：你可以用一些较小的文件，并且在每个文件非注释的第一行都使用 package main 来指明这些文件都属于 main 包。如果你打算编译包名不是为 main 的源文件，如 pack1，编译后产生的对象文件将会是 pack1.a 而不是可执行程序。这样的包称为非 main 包。

# 6. 基本语法---控制结构

# 1. if 语句

与 for 语句一样，if 语句可以在条件表达式前执行一个简单的语句。该语句声明的变量作用域仅在 if 之内。与 js 不同的是，go 语言中的 if 语句不需要使用小括号包裹条件表达式。

在 Go 语言中，if 语句的语法如下：

if 布尔表达式 {
    /* 在布尔表达式为 true 时执行 */
}
