# 一、 vite 简介

# 1. 为什么 vite 启动项目更快

vite 从依赖打包和源码打包这两个方面提升性能

  • 依赖：使用esbuild进行依赖预打包，esbuild使用 Go 语言编写，会比JavaScript-based的打包工具快 10-100 倍。
  • 源码：使用浏览器原生es module提供源码，让浏览器接管打包工具的部分工作。

# 2. vite 热更新

  • 使用 ESM 不需要重新编译：一些打包工具的开发服务器在文件更改时，需要重新国检整个项目，来获取新的模块依赖关系。
  • 使用浏览器缓存加速：vite 利用HTTP头来加速整个页面的重新加载。

# 4. 配置 vite 自动按需引入 vant 组件

# 4.1. 安装插件

npm i unplugin-vue-components -D
1

# 4.2. 配置 vite.config.js

import Components from "unplugin-vue-components/vite";
import { VantResolver } from "unplugin-vue-components/resolvers";
1
2

# 4.3.配置 vite.config.js

export default defineConfig({
  plugins: [
    Components({
      resolvers: [VantResolver()],
    }),
  ],
});
